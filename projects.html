Plotto

Plotto is an algorithm for generating plots for pulp novels created by William Wallace Cook in 1928.
I would like to automate the algorithm even further.

Peddlr: A Bike Loaning App

The Edmonton Bicycle Commuters Society used to run a program that let users access bicycles for free.
I would like to write an application for smart phones that allows users to sign out bicycles, and tracks where they ride them while they're being borrowed.

Think of Pogo, but greener and cheaper.

Help A-Hoye

I currently live with a 70-year-old and a 91-year-old. They're not too confident with how to use their computers, so I help them out.
I explain things like how to save your work in Microsoft Word, or how to make posts on Facebook.
I'm patient and understanding, and I know that it can feel overwhelming to keep up with the breakneck pace of it all.
If you're feeling swept away on the digital tide, I'd be happy to help you out, too.
